คู่มือเริ่มใช้งานแบบสั้น
========================

ไฟล์นี้เป็นแพ็กเกจพร้อมใช้ มี 2 ส่วน
1) public/index.html = หน้าเว็บพร้อมใช้งาน
2) server.js = เซิร์ฟเวอร์กลางแบบ HTTPS สำหรับใช้ในวงแลน

ก่อนรัน
- ติดตั้ง Node.js LTS
- สร้าง cert ด้วย mkcert ให้เรียบร้อย
- ถ้า path cert ไม่ใช่ C:/Users/thx_l/192.168.1.249.pem ให้แก้ใน server.js หรือใช้ environment variables

วิธีรัน
1) เปิด terminal ในโฟลเดอร์นี้
2) npm install
3) npm start
4) ที่คอมหลักเปิด https://localhost:3000
5) ที่มือถือเปิด https://IP-คอมหลัก:3000

เชื่อมต่อเซิร์ฟเวอร์กลาง
- ถ้าเปิดจากคอมหลักที่รันเซิร์ฟเวอร์อยู่ ระบบจะเชื่อมต่ออัตโนมัติ
- ถ้าเปิดหน้าเว็บจากที่อื่น ให้กรอก URL ของคอมหลัก เช่น https://192.168.1.249:3000 แล้วกด เชื่อมต่อ

การเปิดกล้อง
- ต้องเปิดผ่าน HTTPS หรือ localhost
- กด เปิดกล้อง
- อนุญาต camera ครั้งแรก
- กด ปิดกล้อง เมื่อไม่ใช้งาน

# 📸 Tattoo Registry System (ระบบถ่ายภาพกำลังพล)

ระบบสำหรับถ่ายภาพกำลังพล 4 มุม (หน้า / หลัง / ซ้าย / ขวา)
เพื่อบันทึกข้อมูลรอยสัก และจัดทำเอกสารลงนาม

---

# 🎯 จุดประสงค์

* ใช้ถ่ายภาพกำลังพลอย่างเป็นระบบ
* ลดความผิดพลาดหน้างาน
* ใช้งานได้หลายอุปกรณ์พร้อมกัน (มือถือ / คอม)

---

# 🧑‍💻 เหมาะสำหรับใคร

* เจ้าหน้าที่ที่ใช้งานคอมพิวเตอร์ทั่วไป
* ไม่จำเป็นต้องมีพื้นฐานเขียนโปรแกรม

---

# 🧰 สิ่งที่ต้องมี

* คอมพิวเตอร์ 1 เครื่อง (เป็นเครื่องหลัก)
* มือถือหรือคอมอื่น (สำหรับถ่ายภาพ)
* อินเทอร์เน็ต (ใช้ตอนติดตั้งครั้งแรก)
* Wi-Fi (ใช้งานจริงในหน่วย)

---

# ⚙️ ขั้นตอนติดตั้ง (ทำครั้งเดียว)

## 🔹 1. ติดตั้งโปรแกรมพื้นฐาน

### ✅ 1.1 ติดตั้ง Node.js

👉 ดาวน์โหลดที่: https://nodejs.org

ให้เลือก **LTS (Recommended)**

วิธีติดตั้ง:

* เปิดไฟล์ที่โหลดมา
* กด Next → Next → Next ไปเรื่อย ๆ
* กด Finish

📌 ไม่ต้องตั้งค่าอะไรเพิ่ม

---

### ✅ 1.2 ติดตั้ง Git

👉 ดาวน์โหลดที่: https://git-scm.com

วิธีติดตั้ง:

* เปิดไฟล์
* กด Next ไปเรื่อย ๆ
* ใช้ค่าปกติทั้งหมด

---

## 🔹 2. ดาวน์โหลดระบบ

### เปิด Terminal (PowerShell)

📌 สำหรับ Windows 11:

1. กดปุ่ม Start
2. พิมพ์:

```
PowerShell
```

3. คลิกเปิด

👉 ใช้ **โหมดปกติ (ไม่ต้อง Run as Admin)**

---

### ดาวน์โหลดโปรเจกต์

พิมพ์คำสั่งนี้:

```bash
git clone https://github.com/m4maaa/tattoo_registry.git
```

จากนั้นเข้าโฟลเดอร์:

```bash
cd tattoo_registry
```

---

## 🔹 3. ติดตั้งระบบ

พิมพ์:

```bash
npm install
```

📌 รอประมาณ 1–3 นาที

---

## 🔹 4. ตั้งค่า HTTPS (สำคัญมาก)

👉 เพื่อให้มือถือ “เปิดกล้องได้”

---

### 4.1 ติดตั้ง mkcert

พิมพ์:

```powershell
winget install --id FiloSottile.mkcert
```

📌 ถ้ามีถาม ให้กด Yes

---

### 4.2 ปิด PowerShell แล้วเปิดใหม่

---

### 4.3 หา IP เครื่อง

พิมพ์:

```powershell
ipconfig
```

ดูตรงนี้:

```
IPv4 Address: 192.168.x.x
```

📌 จำเลขนี้ไว้

---

### 4.4 สร้าง certificate

```powershell
mkcert -install
mkcert 192.168.x.x
```

---

## 🔹 5. แก้ไฟล์ server.js

เปิดไฟล์:

```
server.js
```

หาโค้ดนี้:

```js
key: fs.readFileSync('...'),
cert: fs.readFileSync('...'),
```

แล้วแก้ให้ตรงเครื่องคุณ เช่น:

```js
key: fs.readFileSync('C:/Users/ชื่อคุณ/192.168.1.10-key.pem'),
cert: fs.readFileSync('C:/Users/ชื่อคุณ/192.168.1.10.pem'),
```

---

## 🔹 6. เปิดระบบ

```bash
npm start
```

---

# 🚀 วิธีใช้งาน

## 🖥️ เครื่องหลัก (คอม)

เปิดเว็บ:

```
https://localhost:3000
```

---

## 📱 มือถือ / เครื่องอื่น

1. ต่อ Wi-Fi เดียวกัน
2. เปิด:

```
https://IP-เครื่องหลัก:3000
```

เช่น:

```
https://192.168.1.10:3000
```

📌 ครั้งแรกต้องกด “ยอมรับ”
